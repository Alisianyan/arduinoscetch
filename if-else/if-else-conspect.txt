Заранее объявим переменную value. Туда можно складывать показания датчика. Допустим, переменная value может изменяться от 0 до 100

Кейс 1. Зажигать светодиод, если значение переменной превысит 50
if (value > 50) {
  зажечь светодиод
}
else {
  выключить светодиод
}

Кейс 2. 
Задача: при значении от 0 до 30 зажигать зелёный светодиод, при значении от 31 до 60 - жёлтый, а при значении от 61 до 100 - красный

тогда следует использоваться следующую конструкцию

if (value <= 30) {
  зажечь зелёный светодиод
}
else if (value <= 60) {
  зажечь жёлтый светодиод
}
else {
  зажечь красный светодиод
}
Выполнение кода идёт от проверки первого условия (value <= 30) до последнего. Здесь у нас всего два условия - (value <= 30) и (value <= 60). Если ничего из этих условий не подошло - тогда выполняется блок else
